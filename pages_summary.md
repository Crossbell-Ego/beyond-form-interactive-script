# 《化形之外》遊戲分頁與 UI 互動邏輯總整理

本專案為網頁端互動劇本殺遊戲 MVP，旨在呈現「拯救爆肝地獄！新人的部門改造計畫」主題。以下為目前的四個分頁、結局畫面、API 設定彈窗及核心樣式系統的完整整理。

---

## 1. 核心樣式系統 (CSS Design System)

* **色彩定義 (`:root`)**：
  * **背景深色**：`#070913`
  * **卡片半透明底**：`rgba(15, 20, 35, 0.65)` (搭配 `backdrop-filter: blur(16px)` 實現磨砂玻璃質感)
  * **夏凌主題色 (HR稽核)**：`#ff3b6c` (冷酷紅，對應漸層 `var(--grad-hr)`)
  * **蘇恬主題色 (總經理)**：`#00f0ff` (改革青，對應漸層 `var(--grad-gm)`)
  * **系統/玩家主色**：`#8b5cf6` (活力紫，對應漸層 `var(--grad-button)`)
* **微動畫集錦**：
  * **漂浮 (`float`)**：立繪輕微上下飄動，增加生動感。
  * **脈衝呼吸 (`pulse`)**：提示文字呼吸變焦。
  * **網格滾動 (`bgPan`)**：首頁背景格線的斜向流動效果。
  * **打字點跳動 (`loaderDotsBlink`)**：AI 思考時三個圓點交替放大淡入。

## 2. 四大分頁結構與設計

### 📌 分頁一：準備畫面 (`#page-start`)
* **定位**：遊戲的啟動首頁。
* **畫面構成**：
  * 主標題「**化形之外 BEYOND FORM**」帶有黑邊陰影與科技感立體字效。
  * 副標題「世界將重新命名當記錄者被記錄之日」。
  * 背景帶有網格流動動畫的漸層遮罩。
* **互動**：在畫面任意處點選「滑鼠左鍵」，觸發淡入動畫切換至 **分頁三**。

### 📌 分頁二：故事背景頁 (`#page-story`)
* **定位**：世界觀與部門困境介紹。
* **畫面構成**：
  * 磨砂玻璃卡片置中。
  * 左側點綴「改革青」邊條的卡片標題。
  * 冷酷紅色的劇本經典台詞引用框：「在這裡，準時下班是一種罪過，瘋狂內鬥是唯一的生存法則。」
* **互動**：在畫面任意處點選「滑鼠左鍵」，切換至 **分頁二**。

### 📌 分頁三：遊戲對話頁 / 開場白 (`#page-game`)
* **定位**：故事的序幕，引出夏凌與蘇恬對新人的首輪質詢。
* **畫面構成**：
  * **人物立繪**：雙方立繪交替置中顯示在螢幕中央。
  * **對話框**：底部傳統 AVG 對話框，帶有 skewed (傾斜) 的角色名稱欄與打字機特效對話內容。
* **互動**：
  * 點擊畫面任意處繼續對話或跳過打字機。
  * 對話順序：夏凌警告 $\rightarrow$ 蘇恬引導 $\rightarrow$ 播畢後自動跳轉至 **分頁四**。

### 📌 分頁四：循環對峙與對答頁 (`#page-chat`)
* **定位**：核心遊戲環節，結合 Gemini API/本地模擬與雙 Agent 交互連鎖對話。
* **畫面構成**：
  * **左右雙立繪**：
    * 尺寸微調至 `260px` 寬、`85%` 高度，釋放空間以避免遮擋。
    * **發言高亮**：發言角色亮度 100% 並帶有主題色外發光特效；未發言者亮度降至 45%。
  * **懸浮透明聊天流**：移除卡片底板，讓對話泡泡直接懸浮在暗化背景圖上。
  * **氣泡對齊與方形名字標籤**：
    * **夏凌（HR）**：靠左對齊。名字為「冷酷紅細框 + 半透明底」的方形小標籤，對話內容換行顯示在名字下方。氣泡為左下尖角。
    * **蘇恬（GM）**：**靠右對齊**。名字為「改革青細框 + 半透明底」的方形小標籤，對話內容換行顯示在名字下方。氣泡為右下尖角。
    * **玩家（你）**：**置中對齊**，且寬度調整為 `96%`（幾乎填滿 750px 對話區）。氣泡為漸層紫色對稱圓角，隱藏「你」名稱，版面更大氣。
  * **膠囊型聊天欄**：
    * 底部單行 `<input>` 輸入框與紫色圓形按鈕（帶有右上方旋轉的實心紙飛機 SVG 圖示）。
    * 支援 **Enter 鍵** 快速送出對策。
  * **思考狀態條**：AI 分析時，輸入框會被等高、等寬的膠囊型狀態條替代，呈現三個跳動點與提示文字。
* **玩法機制**：
  * 玩家輸入對策送出 $\rightarrow$ 夏凌施壓 $\rightarrow$ 蘇恬護航並引導玩家.
  * 對答進行到第 3 輪時，蘇恬發表認可台詞（「很好，既然你提出了這個部門改造計畫...」）並引導切換至 **分頁五**。

### 📌 分頁五：第一章過渡頁 / 劇情懸念 (`#page-transition`)
* **定位**：第一章通關過渡，引出主線科幻/懸疑背後黑幕。
* **畫面構成**：
  * **視覺設計**：全黑背景疊加（`transition-bg-dim`），不顯示任何角色立繪。
  * **文字特效**：畫面上動態依序漸顯浮現【第一章 結束】的劇情檔案內容。
  * **核心懸念**：揭示玩家在深夜加班關機時意外發現的「**吞日計畫**」絕密郵件。
* **互動**：
  * 點擊滑鼠左鍵，切換至 **分頁六：第二章開場對話頁**。

### 📌 分頁六：第二章開場對話頁 (`#page-chapter2-start`)
* **定位**：第二章「吞日黑幕」的開端，登場新主管余鋒烈。
* **畫面構成**：
  * 與分頁三（開場對話頁）結構與排版完全相同，角色立繪（`角色三.png`）置中顯示。
  * 底部為經典 AVG 對話框，顯示余鋒烈的名字與其召集玩家加入新專案的台詞，並套用打字機特效。
* **互動**：
  * 點擊畫面任意處會觸發跳過打字機；打字完畢後再次點擊會切換至 **分頁七：第二章對答頁**。

### 📌 分頁七：第二章對答頁 (`#page-chapter2-chat`)
* **定位**：第二章主線任務對答，與直屬主管余鋒烈交鋒。
* **畫面構成**：
  * 排版與分頁四基本一致，左側為余鋒烈立繪（`角色三_分頁七.png`）。
  * 右側與分頁四右邊角色位置對齊，設有兩張神祕卡片：**【機緣卡】**與**【事件卡】**。
  * 對話歷程氣泡與輸入框區域最大寬度限制為 `660px` 居中（為避免與左右兩側立繪/卡片重疊）。
* **互動與玩法**：
  * 玩家輸入策略送出 $\rightarrow$ 余鋒烈分析並給予熱血理想主義的評價與回饋。
  * **機緣卡**：點擊後自動發送對答「*我剛好認識海外分公司的研發組長，我們可以直接拿到數據對接管道！*」，直接觸發余鋒烈的熱血人脈讚賞回饋並通關成功。
  * **事件卡**：點擊後彈出系統流量限速 80% 警報 Modal，鎖定其它輸入，強迫玩家二選一：
    * 選項 A【直接人肉快遞！】：自動送出「管他什麼限速！我直接帶著硬碟搭計程車去分公司...」並觸發余鋒烈的大力讚賞且通關成功。
    * 選項 B【分批慢慢傳輸】：自動送出「既然被限制了頻寬...分批慢慢傳回來。」並觸發余鋒烈踩雷斥責，重新回到一般對答輸入框供重擬策略。

## 3. 其他互動組件

### 🛠️ AI 設定彈窗 (`#api-modal`)
* **功能**：允許玩家填入自己的 Gemini API Key 及模型名稱（預設為 `gemini-3.1-flash-lite`）。
* **機制**：
  * 自動讀取本地的 `.env` 設定檔（如有設置，會顯示綠色已載入徽章）。
  * 若無填入金鑰，系統將自動降級（Fallback）使用本地的智慧特徵比對進行遊戲模擬。
